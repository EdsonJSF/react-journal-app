# Journal App

## Description

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

## Installation

1. Clone repository
2. Install dependencies
3. Configure env.js
4. Configure .env
5. Configure .env.test (optional)
6. Run server

### Yarn

```sh
yarn
cp env.example.js env.js
yarn dev
```

### NPM

```sh
npm install
cp env.example.js env.js
npm run dev
```
